## 👋 About Me

Hi, I'm **Saurav Premkumar** — an AI/data engineer with 4 years of experience across analytics, BI, data systems, and AI-adjacent workflows, plus an M.S. in Information Systems from **NYU Stern & Courant**.

I like building practical AI systems that connect language models with real tools, data, APIs, and business workflows. My recent work focuses on **LLM applications, RAG, MCP servers, agentic tooling, data pipelines, analytics engineering, and AI-powered automation**.

I’m especially interested in the space where **data gives structure to messy problems, and AI helps turn that structure into usable products**.

![openFDA](https://img.shields.io/badge/-openFDA-2563EB?style=flat&logoColor=white)

Built a FastMCP server that exposes FDA pharmaceutical data to MCP-compatible LLM clients. The server provides tools for official FDA drug labels and adverse event reports, allowing an LLM to route user questions to the right clinical data source.

- Exposes `lookup_drug` and `get_drug_adverse_events` tools through MCP
- Connects LLM clients to FDA label and FAERS adverse-event data
- Demonstrates tool schemas, docstring-driven routing, and agent-style tool use
